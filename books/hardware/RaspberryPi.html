安装最新版本的stretch mini版本，如果没有显示器，没有usb键盘的话，还真的需要两把刷子才能打开ssh功能。该方法除了可以启动ssh服务外，还可以进行软件安装，系统环境设置等等。 把系统dd到microsd卡之后，插卡开机，等待网卡的led灯稳定闪烁，这时候就是系统初始化完成了关机。

sudo apt-get install qemu-user-static binfmt-support
sudo mount /dev/sda2 /mnt
mount --bind /dev dev/
mount --bind /sys sys/
mount --bind /proc proc/
mount --bind /dev/pts dev/pts
cp /usr/bin/qemu-arm-static /mnt/usr/bin
chroot /mnt bin/bash
# or
systemd-nspawn -D /mnt
systemctl enable ssh
